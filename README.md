## Configurar database
* Mariadb 
 ```
 sudo dnf install python3-devel
 ```
 ```
 sudo dnf install mariadb-devel
 ```
 ```
sudo dnf install mariadb-connector-c-devel
 ```
 * Mysql
 ```
 sudo dnf install mysql-connector-python3
 ```
 * Postgresql
 ```
 sudo dnf install libpq-devel
 ```
